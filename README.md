# MarkdownSyntax
Markdown常见语法

|    元素    | Markdowm 语法                                                   |
| :------: | :------------------------------------------------------------ |
| 1 - 6级标题 | # 1级标题，## 2级标题，...###### 6级标题                                 |
|    斜体    | * 斜体内容 *                                                      |
|    加粗    | ** 加粗内容 **                                                    |
| 斜体 加 加粗  | *** 内容 ***                                                    |
|   引用内容   | >                                                             |
|   无序列表   | -                                                             |
|   有序列表   | 1.                                                            |
|   分割线    | ---                                                           |
|   插入链接   | [链接名称](哔哩哔哩 (゜-゜)つロ 干杯~-bilibili](https://www.bilibili.com/)) |
|   插入图片   | ![ 图片名称 ](11.jpg)                                             |
|   内联代码   | `代码内容`                                                        |
